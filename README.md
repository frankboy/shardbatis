####是什么？
是mybatis的一个拦截器插件，用来拦截mybatis的SQL处理过程，方便开发人员在SQL执行前对SQL内容进行一些处理，例如调整表名（这是原
shardbatis的应用场景），调整参数和条件，SQL语句增强等。

该项目脱胎于code.google.com/p/shardbatis ，但是进行了大幅度的改写，由于原项目只能对表名进行处理，而这个不能完全满足我的需要，
无法对SQL进行深入处理，所以有了它。
